MyAcuvue Product

Customer Rewards Platform for Tracking and Earning Reward Points

View GitHub View Notion

Project Overview

Description: MyAcuvue is a customer rewards platform designed to enable users to earn and track reward points upon purchasing products. The platform helps customers easily manage their accumulated points and redeem rewards through an intuitive user interface.

Key Features:

  • User-friendly interface for customers to view and track their reward points.
  • Real-time updates of points earned from product purchases.
  • Seamless client-side routing for smooth user experience.
  • Secure integration with the backend API for tracking user data.
  • Unit tests written with Jest to ensure frontend stability and reliability.

Target Audience: E-commerce businesses, retail companies, and customers seeking an efficient platform for managing loyalty and rewards programs.

Role: Frontend Developer – Developed and maintained the responsive UI components using React.js, implemented routing, integrated API calls, and collaborated on secure authentication.

Technology: React.js, Node.js, Express.js, MongoDB, Jest
