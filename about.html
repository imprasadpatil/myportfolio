Technical Summary

With nearly 3 years of hands-on experience as a MERN Stack Developer, I have developed and deployed 5+ projects using MongoDB, Express.js, React.js, and Node.js. I am proficient in:

  • HTML5, CSS3, and JavaScript (ES6+)
  • Responsive UI design with Bootstrap and MUI for better user engagement
  • Building reusable React components using Hooks, Redux, and React Router for dynamic single-page applications (SPAs)

On the backend, I specialize in:

  • Building RESTful APIs and integrating third-party services
  • Managing asynchronous operations with Axios and Fetch
  • Creating scalable and secure backends using Node.js and Express.js
  • Implementing JWT-based authentication/authorization and writing middleware
  • Working with MongoDB and Mongoose for data modeling, validation, and complex aggregation queries

While my core expertise is in development, I am also transitioning into a project management role. I am experienced with:

  • Agile/Scrum methodologies for leading development sprints
  • Coordinating tasks and ensuring the timely delivery of projects
  • Collaborating with teams to optimize workflows and maintain high-quality code

I am familiar with the following tools and practices to ensure smooth development processes:

  • Git-based version control and collaborative development
  • CI/CD processes, Docker, and unit testing for quality assurance
  • Debugging with Chrome DevTools, Postman, and logging tools to resolve frontend and backend issues

As I move into project management, I aim to leverage my technical experience to bridge the gap between development teams and stakeholders. I’m focused on ensuring smooth communication, efficient workflows, and the successful delivery of high-quality software solutions.
